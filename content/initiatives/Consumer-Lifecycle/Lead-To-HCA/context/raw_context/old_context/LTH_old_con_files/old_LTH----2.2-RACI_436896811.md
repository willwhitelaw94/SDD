Here’s a concise, ready-to-copy RACI matrix template. Replace the grey italics with your own tasks and roles.

<div class="table-wrap">

|  |  |  |  |  |
|----|----|----|----|----|
| Task / Deliverable | **Responsible** (does the work) | **Accountable** (owns the outcome) | **Consulted** (provides input) | **Informed** (kept in the loop) |
| *Example: Draft project charter* | Alice (PM) | Will (CEO) | Finance, Legal | Entire SLT |
| *… add next task …* |  |  |  |  |
| *… add next task …* |  |  |  |  |

</div>

**How to use**

1.  **List tasks** down the first column (break large phases into discrete deliverables).

2.  **List roles** (or names) across the four RACI columns.

3.  For each task, assign:

    - **R** = exactly one or several “doers”.

    - **A** = exactly one ultimate owner.

    - **C** = people whose opinion you’ll actively seek.

    - **I** = anyone who just needs updates.

4.  Scan each row and column to make sure every task has one A and at least one R, and that no one is overloaded with too many Rs.
