================================================================================
                              PATH SUMMARY
================================================================================

| Flow | Actor                  | Trigger                    | End State        |
|------|------------------------|----------------------------|------------------|
| 1    | Recipient/Rep          | First login                | Dashboard/Signed |
| 2    | Recipient/Rep          | Sign HCA                   | HCA Signed       |
| 3    | Staff                  | Alternative consent        | HCA Signed       |
| 4    | Staff                  | Post-meeting flag          | HCA Sent         |
| 5    | System                 | Time elapsed               | Reminders sent   |
| 6    | Staff                  | Terminate agreement        | HCA Terminated   |
| 7    | Staff                  | LTH conversion             | HCA Created      |

================================================================================
                           DECISION POINTS
================================================================================

| Decision                          | Options                              |
|-----------------------------------|--------------------------------------|
| HCA state on login                | Draft → Dashboard, Sent → Signing   |
| Management option                 | Self, Self Plus, Fully Managed       |
| Coordinator required              | Yes → Select, No → Skip              |
| Consent method                    | Digital, PDF Upload, Verbal          |
| Send HCA during conversion        | Yes → Sent state, No → Draft state   |
| Termination reason                | 5 predefined + Other                 |
