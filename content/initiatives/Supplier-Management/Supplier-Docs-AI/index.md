# Supplier Docs AI (SDOC)

AI-powered document processing for supplier compliance documents, automating classification, data extraction, and expiry tracking.

## Problem

Supplier documentation requires manual verification and tracking:
- Manual document review and data entry
- Spreadsheet-based expiry tracking
- Reactive compliance management
- No automated validation

## Solution

Multi-stage AI document processing:

1. **Document Classification** — Auto-identify document type
2. **Data Extraction** — Extract key fields (expiry, coverage, ABN)
3. **Validation** — Cross-check against external sources
4. **Expiry Monitoring** — Proactive alerts before documents lapse
5. **Compliance Dashboard** — Centralized status view

## Success Metrics

| Metric | Target |
|--------|--------|
| Document classification accuracy | 90% |
| Data extraction accuracy | 85% |
| Manual review time reduction | 70% |
