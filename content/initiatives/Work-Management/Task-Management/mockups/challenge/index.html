Linear Lisa

Compact Header with Board Tabs

Eliminates sidebar navigation clutter, keeps focus on tasks. Board tabs (My Tasks, Assigned, Team) as minimal text are faster than a sidebar tree.

Linear Lisa

Sparse Table Rows with Priority Dots

Priority dots are scannable and take less space than badge pills. Combined with clean typography, the list breathes.

Linear Lisa

Keyboard Hints at Bottom-Left

Non-intrusive, always-visible reminder of shortcuts without cluttering the UI.

Notion Nate

Side Panel for Edit (Peek Pattern)

Keeps the list visible while editing — no context loss. Better than a modal for frequent edit workflows.

Notion Nate

Property Table (Key-Value Metadata)

Structured, predictable layout for task metadata. Every property is inline-editable. Scales well with additional fields.

Asana Ada

Progress Sections (Do Today / This Week / Later)

Time-based grouping is more actionable than priority grouping for daily work. "What do I do now?" is answered immediately.

Asana Ada

Stats Bar (Overdue/Today/Upcoming Counts)

At-a-glance awareness of task load without scrolling. Three small cards above the list.

Asana Ada

Rich Kanban Cards with Subtask Progress

The right amount of info on a card — title, priority border, assignee, date, subtask bar. Not too sparse, not too dense.

ClickUp Carl

Bulk Action Bar

ClickUp's floating bulk bar is the most actionable — shows selected count with inline action buttons.

ClickUp Carl

Status Bar

Bottom status bar showing task count, active filters, current grouping, and view mode provides constant orientation.

ClickUp Carl

Inline Editing with Multiple Simultaneous Editors

Power users need to edit multiple cells efficiently. Show one popover at a time but allow fast tab-between-fields.
