## Mission

**Empowering older Australians to live independently in the home they love for longer.**

---

## What We Do

Trilogy Care helps Australians self-manage their home care packages (Support at Home funding). We provide personalized care planning, budget administration, and optional coordination services to help clients maximize their care hours and wellbeing.

## Brand Guidelines

### Primary Colors

| Color          | Hex       | Usage                                |
| -------------- | --------- | ------------------------------------ |
| **Navy**       | `#2C4C79` | Headers, text, professional elements |
| **Sky Blue**   | `#64BCEA` | Main brand color, highlights         |
| **Sun Yellow** | `#FEBD33` | Warmth, optimism, accents            |
| **Orange**     | `#E0763C` | Energy, calls to action              |

### Secondary Colors

| Color          | Hex       | Usage                      |
| -------------- | --------- | -------------------------- |
| **Teal**       | `#43C0BE` | Success, positive states   |
| **Dark Teal**  | `#007F7E` | Secondary success          |
| **Light Blue** | `#D6E8FA` | Backgrounds, soft elements |
| **Light Teal** | `#E3F6F5` | Backgrounds, soft elements |

### Alert Colors

| Color        | Hex       | Usage                 |
| ------------ | --------- | --------------------- |
| **Red**      | `#E04B51` | Warnings, errors      |
| **Dark Red** | `#962E32` | Critical alerts       |
| **Green**    | `#4DC375` | Success confirmations |

### Visual Style

- **Warm, friendly illustration style** matching brand icons
- **Simple, flat shapes** with minimal detail
- **Rounded edges**, organic forms
- **Hand-drawn feel** but clean execution
- **Navy outlines** on key elements
- **No text** in generated images

### Brand Icon Themes

| Element                        | Usage                    |
| ------------------------------ | ------------------------ |
| **Heart shapes**               | Care and hero themes     |
| **Overlapping organic shapes** | Community, connection    |
| **Sun yellow**                 | Warmth and optimism      |
| **Teal checkmarks**            | Compliance and success   |
| **Simple house shapes**        | Home care context        |
| **Connected figures**          | Community representation |

### Character: "Captain Care"

Our brand mascot represents the typical Trilogy Care client:

- Friendly older Australian man, 60s-70s
- White/silver hair, white beard
- Warm genuine smile
- Cream or light colored polo shirt

### Documents in Brand Visuals

When depicting documents or screens:

- Sky Blue (`#64BCEA`) header
- Navy (`#2C4C79`) text elements
- Teal (`#43C0BE`) checkmarks for success
- Light Blue (`#D6E8FA`) backgrounds
