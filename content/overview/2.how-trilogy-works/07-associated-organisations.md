---

## CareVicinity

![CareVicinity Logo](/images/our-family/carevicinity-logo.svg)

**Connecting care seekers with independent support workers**

CareVicinity is an Australian platform that connects care seekers with independent support workers for in-home assistance. The service covers various care types including personal care, daily living support, nursing, and allied health services.

|              |                                                               |
| ------------ | ------------------------------------------------------------- |
| **Funding**  | My Aged Care, NDIS, DVA                                       |
| **Model**    | Marketplace connecting care seekers with workers              |
| **Features** | Workers set their own rates, choose clients, manage schedules |

::callout{icon="i-lucide-external-link"}
[Visit CareVicinity](https://carevicinity.com.au/)
::

## Movemart

**Accessibility equipment marketplace**

Movemart is a multi-vendor marketplace designed to streamline accessibility equipment procurement for care recipients, vendors, occupational therapists, and providers. Customers with HCP and NDIS funding can browse and purchase equipment using pre-approved funding.

|              |                                                 |
| ------------ | ----------------------------------------------- |
| **Funding**  | Home Care Package (HCP), NDIS                   |
| **Model**    | Multi-vendor equipment marketplace              |
| **Features** | Pre-approved funding, guaranteed vendor payment |

::callout{icon="i-lucide-external-link"}
[Learn about Movemart](https://www.fsgcapital.com.au/work/movemart)
::

## Cookaborough

![Cookaborough Logo](/images/our-family/cookaborough-logo.svg)

**Local meals from home cooks**

Cookaborough is a platform that connects consumers with local home cooks and food businesses offering ready-made meals. The marketplace allows customers to browse and purchase prepared dishes from independent food entrepreneurs in their area.

|              |                                                |
| ------------ | ---------------------------------------------- |
| **Model**    | Marketplace for home-cooked meals              |
| **Features** | Local food producers, convenient meal delivery |

::callout{icon="i-lucide-external-link"}
[Visit Cookaborough](https://cookaborough.com.au/)
::

## The Vision

These businesses share a common thread: **empowering individuals through choice and connection**. Whether it's choosing your own care worker, finding the right equipment, or enjoying a home-cooked meal, each platform gives people more control over the services they receive.
