# Foundations

Design foundations are the atomic visual decisions that every component builds upon. These values are defined in Figma and implemented via Tailwind CSS.

- [Colours](/design-system/foundations/colours) — Main and supporting palettes, semantic tokens
- [Typography](/design-system/foundations/typography) — Roboto Flex, type scale, heading levels
- [Logos](/design-system/foundations/logos) — Brand marks and usage variants
- [Shadows & Radius](/design-system/foundations/shadows-and-radius) — Elevation and corner rounding
- [Spacing](/design-system/foundations/spacing) — Tailwind spacing scale, shorthands, and common patterns
- [Motion](/design-system/foundations/motion) — Transitions, easing curves, and duration tokens
- [Breakpoints](/design-system/foundations/breakpoints) — Responsive screen sizes
