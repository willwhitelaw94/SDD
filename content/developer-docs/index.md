## Documentation Structure

This documentation follows the [Diátaxis framework](https://diataxis.fr) — four distinct types of documentation, each serving a different need.

| Section | When to use it |
|---------|----------------|
| [Tutorials](/developer-docs/tutorials) | You're learning something new — follow step-by-step |
| [How-to Guides](/developer-docs/how-to) | You need to accomplish a specific task |
| [Reference](/developer-docs/reference) | You need to look up technical details while working |
| [Explanation](/developer-docs/explanation) | You want to understand why things work the way they do |
