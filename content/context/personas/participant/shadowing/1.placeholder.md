## Adding a Session

When you observe participant interactions (e.g., onboarding calls, care plan reviews), create a new file in this folder with the naming convention:

```
participant-session-{number}-{month}-{year}-{initials}.md
```

For example: `participant-session-1-mar-2025-vf.md`

Include:
- Date and duration
- Interaction type observed
- Key observations
- Pain points identified
- Communication patterns
- Opportunities for improvement
