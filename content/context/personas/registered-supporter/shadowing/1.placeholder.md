# Shadowing Sessions

> No shadowing sessions recorded yet for Registered Supporter interactions.

## Adding a Session

When you observe registered supporter interactions, create a new file in this folder with the naming convention:

```
supporter-session-{number}-{month}-{year}-{initials}.md
```

For example: `supporter-session-1-mar-2025-vf.md`

Include:
- Date and duration
- Interaction type observed
- Key observations
- Pain points identified
- Communication patterns
- Opportunities for improvement
