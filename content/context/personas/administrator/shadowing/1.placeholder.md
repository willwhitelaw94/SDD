# Shadowing Sessions

> No shadowing sessions recorded yet for Administrator roles.

## Adding a Session

When you shadow an Administrator, create a new file in this folder with the naming convention:

```
{role}-session-{number}-{month}-{year}-{initials}.md
```

For example: `finance-session-1-mar-2025-vf.md`

Include:
- Date and duration
- Person shadowed (role, not name)
- Key observations
- Workflows observed
- Pain points identified
- Opportunities for improvement
