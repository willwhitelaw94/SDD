# Shadowing Sessions

> No shadowing sessions recorded yet for Bill Processor roles.

## Adding a Session

When you shadow a Bill Processing Officer, create a new file in this folder with the naming convention:

```
bpo-session-{number}-{month}-{year}-{initials}.md
```

For example: `bpo-session-1-mar-2025-vf.md`

Include:
- Date and duration
- Person shadowed (role, not name)
- Key observations
- Workflows observed
- Pain points identified
- Opportunities for improvement
