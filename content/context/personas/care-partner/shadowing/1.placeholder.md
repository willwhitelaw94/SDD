# Shadowing Sessions

> No shadowing sessions recorded yet for Care Partner roles.

## Adding a Session

When you shadow a Care Partner, create a new file in this folder with the naming convention:

```
cp-session-{number}-{month}-{year}-{initials}.md
```

For example: `cp-session-1-mar-2025-vf.md`

Include:
- Date and duration
- Person shadowed (role, not name)
- Key observations
- Workflows observed
- Pain points identified
- Opportunities for improvement
