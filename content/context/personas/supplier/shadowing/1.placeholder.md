## Adding a Session

When you observe supplier interactions (e.g., onboarding, invoice submissions), create a new file in this folder with the naming convention:

```
supplier-session-{number}-{month}-{year}-{initials}.md
```

For example: `supplier-session-1-mar-2025-vf.md`

Include:
- Date and duration
- Interaction type observed
- Key observations
- Pain points identified
- Portal usage patterns
- Opportunities for improvement
