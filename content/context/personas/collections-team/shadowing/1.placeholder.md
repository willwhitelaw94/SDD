# Shadowing Sessions

> No shadowing sessions recorded yet for Collections Team roles.

## Adding a Session

When you shadow a Collections Team member, create a new file in this folder with the naming convention:

```
collections-session-{number}-{month}-{year}-{initials}.md
```

For example: `collections-session-1-mar-2025-vf.md`

Include:
- Date and duration
- Person shadowed (role, not name)
- Key observations
- Workflows observed
- Pain points identified
- Opportunities for improvement
